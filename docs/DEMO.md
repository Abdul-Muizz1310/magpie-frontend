# Demo Script

## Prerequisites

- magpie-backend running (locally or at the deployed URL)
- magpie-frontend running (`pnpm dev`)

## Steps

1. Open the dashboard at `/` — see all configured sources with status indicators.
2. Click a source (e.g., "hackernews") to view run history and items.
3. Navigate to `/heals` to see self-healing PR history with GitHub links.
4. Visit `/demo` for a guided walkthrough of the self-healing flow.
